# dotfiles

## 環境

- Mac OS X 10.11.4 (15E65)
- tmux 2.1
- zsh 5.2
- iTerm2 2.1.4

## イメージ
![image1](https://raw.githubusercontent.com/KawakawaRitsuki/dotfiles/master/image/image1.png)

## 導入
install.shを実行すれば導入が可能です。   
install.shを実行したあとは、一度再起動することをおすすめします（反映のため）。
